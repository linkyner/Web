Atributos globales

Los atributos HTML son palabras clave dentro de la etiqueta de apertura de un elemento que proporcionan información adicional sobre el elemento.

Tienen la forma de nombre="valor" y se colocan justo después de la etiqueta de apertura para modificar el comportamiento o la funcionalidad del elemento.

Sintaxis

  • Los atributos siempre constan de un nombre y un valor.
  • La sintaxis es nombre="valor", por ejemplo, href="https://www.ejemplo.com".
  • El valor del atributo debe escribirse entre comillas, aunque a veces funciona sin ellas, no es una buena práctica.

Ejemplos comunes

  • href: Se utiliza en la etiqueta <a> para especificar la URL de un enlace.
  • src: Se usa en las etiquetas <img> o <video> para especificar la fuente de la imagen o video.
  • alt: Se utiliza en la etiqueta <img> para proporcionar un texto alternativo si la imagen no se puede mostrar.
  • style: Permite añadir estilos CSS directamente al elemento.
  • lang: Especifica el idioma del documento.
  • title: Muestra un texto emergente cuando el usuario pasa el ratón sobre el elemento.
  • type: En la etiqueta <input>, define el tipo de entrada, como texto, email o botón de envío.

Tipos de atributos

  • Obligatorios: Son necesarios para que un elemento funcione correctamente, como href en un enlace.
  • Opcionales: Modifican la funcionalidad predeterminada del elemento.
  • Estándar: Atributos compatibles con muchos elementos HTML.
  • De eventos: Permiten ejecutar scripts en respuesta a eventos específicos, como onclick o onchange
