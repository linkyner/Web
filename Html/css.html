¿Qué es CSS?

CSS son las siglas de Cascading Style Sheets o Hojas de Estilo en Cascada. Es un lenguaje que se utiliza para describir la presentación de documentos escritos en lenguajes de marcado como HTML o XML.

¿Cómo utilizar CSS con HTML?

Para usar HTML con CSS, se pueden integrar los estilos de tres maneras: CSS externo, donde se vincula un archivo . css usando la etiqueta <link>; CSS interno, donde el código CSS se escribe dentro de la etiqueta <style> en la sección <head> del HTML; y CSS en línea, donde se añaden estilos directamente a una etiqueta HTML con el atributo style . Cada método tiene un propósito y uso diferentes, siendo el externo el más recomendado para proyectos grandes.

CSS en línea

este es un claro ejemplo: <p style="color:red">Texto en rojo</p>

CSS interno

<style> p {color:blue;} </style>

CSS externo

Se usa un archivo separado enlazado con <link>.
